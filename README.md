# Senhorita Executiva - Landing Page

Este repositório contém a landing page da loja de roupas femininas **Senhorita Executiva**.  
O objetivo é apresentar a marca, mostrar alguns produtos, informar endereço/contatos e facilitar o primeiro contato das clientes com a loja.

---

## Descrição geral

A página foi pensada para um público feminino que busca roupas sociais/modernas para trabalho e eventos, com uma pegada mais elegante.  
O layout é responsivo, funcionando bem tanto em desktop quanto em celular.

Principais pontos:
- Visual minimalista, chique e feminino
- Seções bem separadas (sobre, benefícios, coleção, infos da loja, contato)
- Destaque para WhatsApp e Instagram
- Mapa com localização da loja

## Seções da página

- **Header / Navbar**  
  Logo, nome da loja e menu com rolagem para as seções.

- **Banner principal (Hero)**  
  Título, subtítulo e um botão chamando para ver a coleção.

- **Sobre a loja**  
  Texto explicando a proposta da Senhorita Executiva + imagem do ambiente.

- **Benefícios / Diferenciais**  
  Cards destacando pontos como atendimento, qualidade e estilo.

- **Coleção / Produtos**  
  Alguns produtos em destaque com foto, nome, descrição curta e preço.

- **A loja (informações práticas)**  
  Endereço, horários, telefone/WhatsApp, Instagram e mapa incorporado.

- **Contato**  
  Formulário simples (nome, e-mail, telefone e mensagem) e opção de falar direto no WhatsApp.

- **Rodapé**  
  Nome da marca e links para redes sociais.

## Tecnologias utilizadas

- HTML5 (estrutura da página)  
- CSS3 (estilos, responsividade e alguns efeitos)  
- JavaScript puro (interações, rolagem suave, menu mobile)  
- Bootstrap 5 (grid e alguns ajustes responsivos)  
- Font Awesome (ícones)
